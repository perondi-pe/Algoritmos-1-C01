# Algoritmos 1 - Exercícios

Este repositório contém os códigos desenvolvidos durante as aulas da disciplina **Algoritmos 1**.

## 📚 Sobre

Aqui estão armazenados exercícios, exemplos e práticas feitos ao longo da disciplina para aprendizado de lógica de programação e desenvolvimento de algoritmos.

Os códigos foram feitos como parte das atividades da aula e servem para estudo e prática.

## 🧠 Conteúdos estudados

Entre os principais tópicos abordados estão:

- Lógica de programação
- Variáveis
- Entrada e saída de dados
- Estruturas condicionais (`if`, `else`)
- Estruturas de repetição (`for`, `while`)
- Vetores / listas
- Funções
- Resolução de problemas com algoritmos
